# Kokoro Note Website Draft

Free static website draft for Cloudflare Pages.

## Edit content
- Home page: `index.html`
- Style/colors: `style.css`
- Test cards: `data.json`
- Quiz page: `tests/renai-izon.html`
- Images: replace files in `assets/images/`

## Deploy to Cloudflare Pages
1. In Cloudflare Pages, connect this GitHub repository.
2. Build command: leave blank
3. Output directory: `/`
4. Deploy.

## AdSense later
This draft includes ad placeholders:
- In `tests/renai-izon.html`, search for `AdSense placeholder`.
- After approval, replace the placeholder div with your AdSense ad unit code.

Before AdSense application:
- Replace Privacy Policy with real policy.
- Add more unique content pages.
- Add your domain in canonical, sitemap, robots.txt.
- Use original or properly licensed images.
